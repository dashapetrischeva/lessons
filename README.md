# HTML, CSS & JavaScript: Comprehensive Core Training

This repository contains practical assignments and code lab projects completed during an intensive 5.5-month frontend development course. The program combined deep theoretical fundamentals with rigorous hands-on practice.

## 📊 Course Overview
* **Duration:** 5.5 months of intensive learning
* **Total Lessons:** 39 comprehensive sessions (24 theoretical modules + 15 dedicated practical workshops)
* **Practical Deliverables:** 18 extensive, reviewed homework assignments

## 🛠️ Technologies & Concepts Covered

### Modern HTML5 & Web Semantics
* Introduction to development environment, workflows, and dev tools
* Semantic HTML structure and content-driven tags (text, media, and forms)
* Image optimization, responsive images, and modern asset handling

### Advanced CSS3, SCSS & Responsive Design
* CSS Syntax, selectors, specificity, and cascading rules
* Typography, custom web font integration, and advanced text styling
* CSS Box Model: properties, dimensions, margins, paddings, and borders
* Visual styling: backgrounds, gradients, shadows, and modern effects
* Interaction: CSS pseudo-classes, pseudo-elements, and smooth CSS animations
* Layout engines: Advanced CSS Positioning, Flexbox, and CSS Grid Layout
* Responsive Web Design (RWD) & Mobile-First development using media queries
* CSS preprocessing with **SCSS/SASS** (variables, nesting, mixins)

### Core JavaScript (ES6+) & DOM Manipulation
* JavaScript syntax, primitives, data types, variables, and memory concepts
* Working with complex data structures: Arrays and modern array methods
* Document Object Model (DOM) manipulation (bridging HTML, CSS, and JS)
* Event handling, dynamic UI rendering, and basic asynchronous interaction

### Testing & Quality Assurance
* Cross-browser compatibility testing
* Code validation, responsive layout testing, and pixel-perfect verification
* Custom group-selected practical implementation tasks

## 📂 Repository Structure
The repository is organized by modules and homework assignments. Each directory showcases clean, semantic markup, modular CSS/SCSS styles, and interactive JavaScript logic.
